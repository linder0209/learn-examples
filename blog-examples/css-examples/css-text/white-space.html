1. normal：

    white-space 设为 normal 默认 显示 的效果

  2. pre：

    不合并文字 间的空白符 （包括换行字符）， 当文字超出边界时不换行， 与标签 pre 的行为是一样的

  3. nowrap：

    强制在 同一行内显示所有文本， 合并文本间的多余空白（包括换行字符）， 直到文本结束或者遇到 br
    元素。

  4. pre-wrap：

    不合并文字间的 空白符 （包括换行字符），当文字超出边界时会自动换行。

  5. pre-line

    合并文字间的 空白符， 但会保留换行符，并且当文字超出边界时会自动换行。
